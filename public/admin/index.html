⚠️ Konfigurasi GitHub Diperlukan

Untuk menggunakan CMS ini, Anda perlu setup GitHub OAuth terlebih dahulu:

1. Edit file /public/admin/config.yml

2. Ganti repo: your-username/diamond-group dengan repo GitHub Anda

3. Setup GitHub OAuth App di GitHub Developer Settings

4. Atau gunakan npx decap-server untuk development lokal
